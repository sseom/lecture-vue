### Routiog 라우팅
- 서버 라우팅 
  - 매번 주소 요청시 화면이 갱신됨 (ex. 네이버, 구글..)
- 브라우져 라우팅 
  - 전체 화면이 갱신되지 않고 필요한 데이터만 서버에 요청해서 화면을 갱신함 (ex. 구글메일, 트렐로..)
  - 주소 요청시 라우팅을 브라우저에서 처리하고 데이터는 백엔드 API를 통해 서버로부터 받아와서 데이터이용해서 화면에 뿌리는 구조
